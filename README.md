# rust-lua53
Aims to be complete Rust bindings for Lua 5.3.

Requires `gcc` and a copy of Lua 5.3 to build. To get started:

    export LUA_INCLUDE=/path/to/lua53/headers
    ./prebuild.sh
    cargo build
